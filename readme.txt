autenticacion
=============

Sistema de autenticacion de usuarios desarrollado en Python con Flask.

Descripcion
-----------
Implementacion de un sistema de autenticacion que cubre el flujo completo de registro e inicio de sesion de usuarios. Incluye manejo de sesiones mediante cookies y autenticacion basada en tokens JWT, con proteccion de rutas para usuarios no autenticados.

Tecnologias
-----------
- Python
- Flask
- JWT (JSON Web Tokens)
- Cookies de sesion

Estructura
----------
- readme.txt  Documentacion del proyecto

Como ejecutar
-------------
1. Instalar dependencias: pip install -r requirements.txt
2. Ejecutar el servidor: python app.py
3. Acceder en: http://localhost:5000
